초보자를 위한 전신운동

바벨 스쿼트 10~15회 X 4set

  1. 발을 어깨 너비로 벌리고 선다. 등의 자연스러운 아치를 유지하고 견갑골을 조이고 가슴을 들어 올리십시오.
  2. 어깨를 가로질러 바를 잡고 등 위쪽으로 지지합니다. 다리를 곧게 펴서 막대를 풀고 한 걸음 뒤로 물러나십시오.
  3. 엉덩이가 무릎 아래에 올 때까지 등의 형태를 바꾸지 않고 무게를 낮출 때 무릎을 구부립니다.
  4. 바를 시작 위치로 되돌리고 다리를 들어 올리며 상단에서 숨을 내쉬십시오.

바벨 벤치 프레스 12~15회 X 4set

  1. 바닥에 발을 대고 벤치에 평평하게 눕습니다. 곧은 팔로 바의 랙을 풉니다.
  2. 바를 가슴 중앙까지 내립니다.
  3. 팔꿈치가 잠길 때까지 바를 올립니다.

친 업 10~15회 X 4set

  1. 바를 어깨너비로 벌리고 앙와위 그립(손바닥이 당신을 향함)으로 잡습니다.
  2. 몸을 매달고 팔을 완전히 뻗은 상태에서 턱이 바를 지나갈 때까지 몸을 위로 당깁니다.
  3. 천천히 시작 위치로 돌아갑니다. 반복하다.

렛풀 다운 8~12회 X 4set

  1. 바를 어깨보다 약간 넓게 잡아줍니다.
  2. 숨을 뱉으며 팔꿈치가 옆구리나 바닥을 찍는 느낌으로 약간 누우며 당겨줍니다.
  3. 숨을 마시며 등의 이완을 느끼며 올라가줍니다.

바벨 오버헤드 프레스 10~15회 X 4set

  1. 대략 어깨 너비의 그립을 잡습니다. 팔꿈치에서 주먹(수직 팔뚝)까지 직선이 있어야 합니다.
  2. 턱을 뒤로 당기고 팔꿈치 관절에서 확장하고 어깨 관절에서 굴곡하여 천장을 향해 무게를 누릅니다.
  3. 팔꿈치가 펴질 때까지 누르고 머리를 앞으로 살짝 밀어줍니다.
  4. 컨트롤을 사용하여 시작 위치로 돌아갑니다. 바가 얼굴을 안전하게 통과할 수 있도록 턱을 뒤로 당깁니다.

바벨 루마니안 데드리프트 10~15회 X 4set

  1. 어깨 너비, 더블 오버핸드 또는 혼합 그립을 취하십시오.
  2. 무릎은 대부분 펴진 상태에서 엉덩이를 뒤로 미십시오. 햄스트링의 스트레칭을 찾으십시오.
  3. 스트레칭을 느끼면 서있는 자세로 돌아올 때까지 엉덩이를 앞으로 밀어냅니다.

  2. 팔꿈치를 무릎 안쪽에 유지하면서 쪼그리고 앉습니다.

포워드 런지 12~15회 X 4set

  1. 한 발로 앞으로 나아갑니다.
  2. 뒤쪽 무릎이 거의 바닥에 닿을 때까지 몸을 낮춥니다.
  3. 똑바로 유지하고 앞 무릎이 앞 발 위에 있는지 확인하십시오.
  4. 시작 위치로 돌아올 때까지 앞발로 바닥을 밉니다. 다리를 바꿉니다.
